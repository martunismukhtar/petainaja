# 🌍 Web GIS Map Builder

Aplikasi web untuk siapa saja yang ingin mengubah data spasial menjadi peta yang dapat dibagikan dan dicetak — **tanpa instalasi dan tanpa memerlukan latar belakang GIS**.

## ✨ Fitur Utama

### 🗺️ Interactive Map

* Halaman peta interaktif menggunakan **OpenFreeMap** dan **Protomaps** sebagai basemap.
* Navigasi peta yang responsif dengan dukungan zoom dan pan.

### 📂 Import Data Spasial

Mendukung upload dan parsing berbagai format data:

* GeoJSON (`.geojson`, `.json`)
* CSV (`.csv`)
* Shapefile (`.shp`)
* KML (`.kml`)

Seluruh data akan dirender secara langsung menggunakan **MapLibre GL JS**.

### 📑 Layer Management

* Menampilkan daftar layer yang aktif.
* Toggle visibilitas layer.
* Mengelola beberapa layer dalam satu proyek.

### 🎨 Style Editor

Melakukan kustomisasi tampilan layer dengan mudah:

* Mengubah warna layer
* Mengatur opacity
* Mengubah ukuran marker
* Pratinjau perubahan secara real-time

### ✏️ Editing

* Edit geometri secara langsung pada peta.
* Tambah, ubah, dan hapus fitur spasial.

### 🏷️ Attribute Editor

* Menambahkan atribut baru.
* Mengubah nilai atribut.
* Menghapus atribut yang tidak diperlukan.

### 📊 Spatial Analysis

Menyediakan berbagai fitur analisis spasial untuk membantu pengguna memahami data secara lebih mendalam.

### 🖨️ Map Layout Composer

Membuat layout peta siap cetak dengan komponen:

* Judul peta
* Legenda
* North Arrow
* Scale Bar

### 🔗 Share Map

* Membuat tautan publik untuk membagikan peta kepada pengguna lain.

### 🖨️ Print Map

* Cetak peta dalam format siap presentasi maupun dokumentasi.

---

## 🚀 Tujuan Project

Project ini bertujuan untuk mempermudah siapa saja dalam membuat, mengedit, menganalisis, dan membagikan peta secara online tanpa harus menggunakan perangkat lunak GIS desktop yang kompleks.

---

## 🛠️ Teknologi

* **MapLibre GL JS** — Rendering peta interaktif.
* **OpenFreeMap** — Basemap terbuka.
* **Protomaps** — Vector basemap.
* **GeoJSON / CSV / SHP / KML Parser** — Import berbagai format data spasial.
